Transform your plain text into static websites and blogs

What is this?

Jekyll is a simple, blog-aware, static site generator for personal, project, or organization sites. Written in Ruby by Tom Preston-Werner, GitHub's co-founder, it is distributed under the open source MIT license

Features

Instead of using databases, Jekyll takes the content, renders Markdown or Textile and Liquid templates and produces a complete, static website ready to be served by Apache HTTP Server, Nginx or another web server Jekyll is the engine behind GitHub Pages a GitHub feature that allows users to host websites based on their GitHub repositories for no additional cost.

Installation

Check that in Official jekyll Documentation, for install jekyll for your working environment

Jekyll Installination - Docs

Create an app

Create a New Jekyll site, By running that command

jekyll new mynewsite

Change your Directory

cd mynewsite

Then run that following command, to launch your website, on a local developement server

bundle exec jekyll serve

It Normally launched in 127.0.0.1 (localhost), port 4000

If you want to change the PORT address, run that following command

bundle exec jekyll serve --port mynewportnumber
